Spring Lake Elementary | Website Project Analysis

Site Intent

The goal of the new Spring Lake Elementary School website is to consolidate the site navigation on the Homepage by separating information into clear sections for quick and easy site access. Site content will be grouped to minimize the number of pages visitors access to find information. The site code will be upgraded to current web standards to allow for faster loading and updating of information. Update the site design to a simple clean design that targets the primary content that parents require.

Competitive Analysis

Spring Lake Elementary School is located in Altmonte Springs FL, and is a part of Seminole County Public Schools. Competitive schools in the area are rated slightly above Spring Lake according to Great Schools. Reviews of Spring Lake point to an opportunity to improve communication. An focused website can aid in improving communication for current and potential parents.

The three public schools closest to Spring Lake elementary are Altamonte Elementary, Forest City Elementary, and Lake Orienta Elementary. In analyzing these school sites. Lake Orienta has the most up to date web standards and design. Closely following is Altamonet Elementary site, which has an updated navigation but the overall site is still outdated. Only one of these sites uses any target keyword in the meta-tag for SEO.

Targeted Audience

The Site demographics from Alexa state that the main target audience is female ages 35-44 with children, has some college education and often browse form home. The site will focus on this target ensuring that the information for parents stands out and is easily accessible.
